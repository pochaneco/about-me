## 私個人について

### ご挨拶
こんにちは。及川と申します。最近はWEB業界でバックエンド寄りのフルスタックエンジニアとしてやっております。神奈川県のちょっと田舎にてフルリモート在宅でやらせていただいております。バックエンドでは特に Laravel にハマってますが、コンポーネント指向などのフロントエンドの最近の考え方も大変興味深いと思い、ReactやVueなど触っております。エンジニアとして11年。大小様々プロジェクトをやらせていただきました。

### 略歴

#### SIer時代
高校でBASICやC言語でのプログラミングの楽しさに目覚め、進学せずSIerに就職し、最初の3年ほどはIBMのAS/400などがメインで、某食肉流通加工の大企業のBIツールや企業情報DBの開発・運用などをし、RPG言語やCOBOL言語でエンジニアとしての礎を築きました。性格のマメさか器用さが良かったのか、大きなプロジェクトで共通基盤チームを任されるなどし、現場でも社内でも大変重宝されました。

#### エンジニアじゃない時代
結婚を期に5年ほどでSIerを多忙かつ低賃金のため辞め、家族のために自分に何ができるかを試すため、いくつかのアルバイトや就職活動を経験しました。飲食や携帯電話のセールスマンなどの接客業も数ヶ月やりました。結果わかったのは、自分にとってエンジニアが一番楽しくて稼げるということ。

#### モバイル業界時代
その後、モバイル業界の某企業に出会い、フリーランスエンジニアとして、バックエンドはPHPやJava、C# 等、フロントエンドもやらせていただき、5年ほど常駐させていただきました。4Dというマイナーな統合環境なども使用したり、主にBtoBのサービスの構築に企画から入るなどし、エンジニアとしての幅を広げました。

#### 現在
ご縁があって、今年度よりWeb業界に転身し、在宅エンジニアと言う前職とはまた違った仕事のやり方や、Web業界での新しい技術に触れ、さらに幅広い業務に対応できるようになっております。

### 仕事への思い
三度の飯よりコーディングが好きです。学生時代に培ったアセンブリ言語やBASIC言語の知識により、コンピュータと会話することを心がけており、「すべてのプログラミング言語は人間とコンピュータとの共通言語という道具」だから「ヒトにとって読みやすくて、コンピュータにとって効率的な処理を、素早く書くこと」を心がけております。

#### 野望
まだまだひよっ子ですが、世界に名だたる日本一のエンジニアを目指して日々精進しております！！最近は他人のソースを読むのが密かな楽しみ。今の夢は、大企業が数十億円＋数年かけてやっていたようなシステムを、少人数・短期間で達成することです。

#### 直近の目標
PHPはそれなりにできるようになったので、RubyとPythonを深めていき、適材適所ができるようになりたいと思っています。また1つサービスを作りたいと思っているものがあり、そのために色々な知識を練っています。
